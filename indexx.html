Reagent Mixture and Level Controlling

Problem Solution

In this example, we use PLC programming of Siemens S7-300 PLC. We can use other PLC also for this application. Two level sensors are used for detecting the level of material A and material B. Also one bottom level sensor used for detecting the bottom level.

To control level of this system, single acting valve is used which has two states, either fully opened or fully closed. Particular is provided to mix the material A and material B in a tank. We can use on delay timer this function.

After successfully completion of mixing, outlet valve is operated to drain the mixed material. When mixing process is completed, buzzer will be activated and it will remain ON and after 5sec it will be automatically OFF.

PLC Logic Explained

For this application, we used S7-300 PLC and TIA portal software for programming.

Network 1: System Latching

Latching circuit for the cycle. START PB (I0.0) sets Cycle ON (Q0.0), and STOP PB (I0.1) resets it.

Network 2: Material A Intake

Inlet valve 1 (Q0.1) activates when bottom level (I0.2) is detected and Material A level (I1.0) is not yet reached.

Network 3: Material B Intake

Inlet valve 2 (Q0.2) activates once Material A is full and Material B level (I1.1) is not detected.

Network 4 & 5: Agitation Process

When tank is full (I1.1 detected), Agitator Motor (Q0.3) runs for 20 seconds via timer instruction.

Network 6: Draining Process

Once mixing timer completes, the Outlet Valve (Q0.4) opens to drain the material.

Network 7, 8 & 9: Alarm Notification

Completion triggers the Buzzer (Q0.5). It remains active for 5 seconds before the ON delay timer automatically resets the output.
